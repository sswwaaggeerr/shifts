# 📅 index.html — Фронтенд-шаблон графика смен

📩 *Файл подготовлен в рамках хакатона по кейсу автоматизации графиков работы сотрудников.*  
🔗 [Шаблон для просмотра](https://sswwaaggeerr.github.io/shifts/)

> Полностью автономный интерфейс для планирования, согласования и экспорта рабочих смен.  
> Открывается в браузере без установки сервера. Идеально для демо и хакатона.

## 🎯 Что это такое
`index.html` — это **готовый пользовательский интерфейс** (HTML + CSS + Vanilla JS), который имитирует работу системы планирования графиков. Все данные хранятся локально в браузере (`localStorage`), поэтому файл работает сразу после скачивания, без настройки бэкенда или базы данных.

## ✨ Возможности из коробки
| Функция | Статус |
|--------|--------|
| 🎭 Переключение ролей | ✅ Сотрудник / Руководитель |
| 📅 Планирование на 14 дней | ✅ С валидацией пересечений и лимитов |
| 📊 Таблица графиков | ✅ Фильтры: все / запланированные / согласованные / неявки |
| 📥 Экспорт в CSV | ✅ Корректно открывается в Excel (UTF-8 + BOM) |
| 💾 Сохранение данных | ✅ Автоматически в `localStorage` браузера |
| 📱 Адаптивность | ✅ Работает на мобильных и десктопе |

## 🚀 Как открыть и использовать
1. Скачайте файл `index.html`
2. Дважды кликните по нему или перетащите в окно браузера (Chrome, Edge, Firefox, Safari)
3. Выберите роль (`👤 Сотрудник` или `👑 Руководитель`)
4. Добавляйте смены, фильтруйте список, нажимайте `📎 Выгрузить CSV`

> 💡 Данные привязаны к вашему браузеру. Если откроете файл на другом устройстве или в режиме инкогнито, они не подтянутся.

## ⚠️ Текущие ограничения
- 🔒 Нет авторизации и серверной части (все проверки выполняются на клиенте)
- 🗃️ Организационная структура (`alliancesData`) захардкожена в коде
- 🔄 Нет синхронизации между пользователями (данные видны только тому, кто их создал)

## 🔗 Следующие шаги (для production)
Файл **готов к интеграции** с Flask/Python-бэкендом:
1. Переместите в папку `templates/` и переименуйте в `index.html`
2. Замените `alliancesData` на `fetch('/api/alliances')`
3. Замените `localStorage` на `fetch('/api/shifts')` (GET/POST/DELETE)
4. Привяжите кнопку экспорта к `window.location.href = '/api/export'`
5. Добавьте передачу роли из Flask: `const CURRENT_ROLE = "{{ current_user.role }}";`

## 🛠 Технологии
- `HTML5` + `CSS3` (встроенные стили, без внешних зависимостей)
- `Vanilla JavaScript` (без jQuery/React/Vue)
- `Web Storage API` (`localStorage`)
- `Blob API` + `URL.createObjectURL` (генерация CSV на клиенте)
